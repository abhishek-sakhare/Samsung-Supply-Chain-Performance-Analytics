# Samsung Supply Chain & Logistics Analytics Dashboard

## Overview

The **Samsung Supply Chain & Logistics Analytics Dashboard** is an end-to-end Business Intelligence project developed using **Power BI**. The dashboard provides comprehensive insights into Samsung's supply chain operations, covering procurement, inventory management, warehousing, logistics, and sales performance.

The project demonstrates the complete analytics workflow, including data ingestion, ETL, data modeling, DAX measure creation, and interactive dashboard design. It enables stakeholders to monitor operational performance, identify bottlenecks, and make data-driven business decisions.

---

## Problem Statement

Managing a large-scale supply chain involves coordinating suppliers, inventory, warehouses, shipments, and sales across multiple locations. As operational data grows, it becomes increasingly difficult to monitor performance, identify delays, and optimize inventory using raw datasets.

The objective of this project is to develop an interactive Power BI dashboard that consolidates supply chain data into a single analytical platform, enabling stakeholders to monitor key performance indicators, identify inefficiencies, and improve overall supply chain performance.

---

## Objectives

* Build an end-to-end Supply Chain Analytics Dashboard using Power BI.
* Perform ETL and data transformation using Power Query.

## Data Analytics Workflow

### 1. Data Ingestion

* Imported multiple datasets using Power Query.
* Implemented folder-based data loading.
* Created parameterized file paths for portability.

### 2. Data Cleaning & Transformation

* Removed duplicate records.
* Handled missing values.
* Standardized data formats.
* Renamed columns.
* Created calculated columns.
* Prepared clean datasets for reporting.

### 3. Data Modeling

* Designed a Star Schema architecture.
* Established one-to-many relationships.
* Optimized the model for accurate filtering and high performance.

### 4. DAX Development

Created reusable business measures including:

* Revenue
* Profit
* Profit Margin
* Inventory Turnover
* Average Lead Time
* Perfect Order Percentage
* Shipment Delay Rate
* Defect Rate

### 5. Dashboard Design

* Business-oriented dashboard layout.
* Consistent theme and formatting.
* Interactive navigation.
* Dynamic visualizations for detailed analysis.
